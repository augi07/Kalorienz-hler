# Kalorienzähler

Installation des node_modules bitte im Ordner Kalorienzähler. Um Node_modules zu installieren:

npm install

Anschliessend im Terminal auch:

npm run dev

mit http://localhost:9000 auf die Kalorienzähler Seite gehen
